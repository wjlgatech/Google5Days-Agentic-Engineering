Module · M1 — Your First Agent Loop

A real, runnable, zero-dependency agent. The 7-part learning structure, adapted for all 5 personas. The template every future module copies.
7 parts × 5 personas · runnable

Module · M2 — Give Your Agent Hands

Tools with contracts. A zero-dep tool registry that validates every call and returns errors that teach recovery. Day 2, made runnable, for all 5 personas.
tools · 8-field contract · runnable
🧠

Module · M3 — Give Your Agent Memory

Context engineering: sessions + long-term memory. A zero-dep context manager — compaction, blended retrieval, provenance, and a budget that beats context rot. Day 3, for all 5 personas.
sessions · memory · runnable
⚖️

Module · M4 — Is Your Agent Any Good?

Agent quality: evaluate the trajectory, not just the answer. A zero-dep evaluator — Outside-In scoring, the Four Pillars, pairwise judging, and a human gate it never self-answers. Day 4, for all 5 personas.
trajectory eval · 4 pillars · runnable
🚀

Module · M5 — Ship It Without Breaking It

Prototype → production (AgentOps). A zero-dep evaluation-gated deploy pipeline that imports M4 as its gate: blocked → canary → human sign-off → promoted, then evolves failures into tests. Day 5, for all 5 personas. Closes the loop.
eval-gated deploy · canary · runnable
🧰

Tool · spec-to-green

Turn a fuzzy request into a verifiable spec with a deterministic check, then drive code to green. Full 8-field contract; portable across Claude Code, Codex, Hermes, Gemini.
8-field contract · portable

The hub is a living system: every run records a decision and a lesson, so the next run starts smarter. Correctness is enforced deterministically (bash scripts/check.sh) — but “good?” stays a human call.
