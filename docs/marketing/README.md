# Marketing

Narrative assets for the Agentic Engineering Hub. Both pieces connect the hub to the thesis that
sparked it — [William Wu's FDE-os article](https://www.linkedin.com/pulse/fastest-growing-job-ai-has-school-so-im-building-one-learns-wu-phd-uoasc/)
— and are grounded in cited sources.

| File | What it is | Length | Audience |
|---|---|---|---|
| [`ARTICLE-from-prompt-to-proof.md`](ARTICLE-from-prompt-to-proof.md) | Long-form article — fun enough for a 15-year-old, sharp enough for an AI director, fully referenced | ~1,400 words | blog / LinkedIn article / docs |
| [`SHORT-POST.md`](SHORT-POST.md) | Punchy social post | ~190 words | LinkedIn / X / newsletter |

**Core message:** the unit of AI work moved from the *prompt* to the *evaluated loop*; this hub is
a runnable school for that loop, built from Google's 5-Day Agentic Engineering whitepapers, and it
proves itself (`bash scripts/check.sh` → 81 green, incl. a live end-to-end test).

**Every claim is reproducible or cited.** Hub claims (81 checks, runnable modules, live e2e) →
run the harness. External claims (MIT 95%, FDE 1,165%) → see the Sources section of the long-form
article. Primary sources (the five whitepapers) are in [`docs/`](../).
