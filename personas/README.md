# Personas — plug in at your level

The hub adapts to **who you are and what you're responsible for**. Same substrate, five
on-ramps (Design Principle #3: progressive complexity; Day 1 Levels 0–4). Pick the closest —
you can switch any time, and every module is written for all five.

| Persona | You care most about | Start |
|---|---|---|
| 🧒 **15 — Explorer** | confidence, curiosity, a fun working project today | [`15-explorer.md`](15-explorer.md) |
| 🛠️ **20 — Junior Engineer** | workflows, coding-agent collaboration, debugging, shipping | [`20-junior.md`](20-junior.md) |
| 🏗️ **30 — Senior Engineer** | system design, reliability, reusable platform thinking | [`30-senior.md`](30-senior.md) |
| 🎯 **40 — AI Director** | operating models, evaluation, risk, transformation | [`40-director.md`](40-director.md) |
| 💼 **50 — EVP / Executive** | strategy, investment, governance, measurable impact | [`50-executive.md`](50-executive.md) |

**Rule of thumb:** start one level *below* where you think you are. Faster win, same depth
underneath. Every persona's first stop is the same module — **[M1 · Your First Agent
Loop](../modules/M1-your-first-agent-loop/)** — just framed for you.

➡️ **See the whole journey:** [**Learning Paths (M1 → M5, per persona)**](LEARNING-PATHS.md) — the
capstone map that threads all five modules into one path for each persona, each ending in an
integrative project and a measurable "graduated when."
